it could directly be taken as the sample 

## Clone with Submodules

This repository uses Git submodules. Make sure to clone it with:

```bash
git clone --recurse-submodules https://github.com/satjhariya/stm32f4xx_hal_app.git
```

If you already cloned the repository without submodules, run:

```bash
git submodule update --init --recursive
```
